```json
    }

}
```

## 参考图 UI 分析

- 全局风格：深色专业工具界面，强调画布与侧边功能面板，不做营销式首页。
- 布局：左侧竖向图标工具栏，中央大画布，右侧为 Tab/词库/知识图谱等操作区。
- 标注形态：画面上密集悬浮术语气泡，标注与图像元素直接关联。
- 工具模式：矩形标注、钢笔标注、SAM 标注并列出现，工具入口图标化。
- 数据目标：支持多维度、多层次标注，导出类 Git 版本管理和 AI 训练格式，后期需要 `.iiml` / `.hpsml`。
- 图谱目标：右侧节点-边知识图谱，橙色节点、中文标签、点击回联左侧图像元素。
