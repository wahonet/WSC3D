# P2-A 拼接模块开发记录

## 已实现

- 独立 Three.js 拼接场景，与 P1 浏览场景隔离。
- 右侧拼接控制面板：添加画像石、已加载列表、A/B 角色选择、锁定、移除。
- 简化后的默认流程：进入拼接模式后自动加载当前画像石，选中单块后在画布中出现微调浮动面板。
- 微调浮动面板支持 1/5/10cm 步长平移、X/Y/Z 三轴 5° 旋转、选中石块复位。
- 最多加载 10 块画像石模型。
- 每块模型按 AABB 预计算 6 个可选面：上、下、左、右、正、背。
- 默认点击场景只选中石块用于微调，不再进入 A/B 选面。
- A/B 面吸附已收进“高级选边吸附”；只有点击“选择 A 面/选择 B 面”后才进入一次选面模式。
- AABB 面只用于内部拾取和吸附计算，不再在三维视图中显示红/蓝透明面，避免误认成文物结构。
- 执行拼接时：
  - 计算 A 面世界中心点与法向量。
  - 计算 B 面世界中心点与法向量。
  - 旋转 B，使 B 面法向量与 A 面法向量反向。
  - 平移 B，使两面中心重合。
  - 更新 B 的位姿 transform，后续可保存为方案 JSON。
